Forecasting Delivery Time with Proximity-Based Notifications

Tools & Libraries

  • Python, Pandas, NumPy
  • Scikit-learn (model training)
  • Geopy / Haversine (distance)
  • Flask (API)
  • SQLite or PostgreSQL
  • HTML + Chart.js/JS (Frontend)

By the end of this module, interns will:

  • Clean and prepare historical delivery datasets.
  • Train an ETA prediction model using linear regression or random forest.
  • Calculate real-time distance using coordinates.
  • Create a Flask API to return ETA predictions and proximity notifications.
  • Display live delivery updates and ETA on a basic frontend.
